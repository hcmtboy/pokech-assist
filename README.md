# PokeCh Assist

OBSで表示しているNintendo Switch対戦画面の横に置く、ポケモン対戦補助ツールのプロトタイプです。

## 使い方

`index.html` をブラウザで開きます。OBSに入れる場合は「ドック」または「ブラウザソース」でこのHTMLを指定します。

`file://` で直接開いた場合は内蔵データで動きます。ローカルサーバーで開いた場合は `data/*.json` を読み込みます。

現時点では手入力版です。

- 相手ポケモンを選択すると、ステータス、素早さ比較、タイプ相性、採用率サンプルを表示します。
- NNで相手名が判別できない場合も、敵6体登録から本来のポケモン名を選べます。
- 敵6体を登録すると、相手ポケモンの選択欄は登録済みの6体だけになります。
- 味方6体を登録すると、味方ポケモンの選択欄は登録済みの6体だけになります。
- 初期表示はライトテーマです。右上のボタンでライト/ダークテーマを切り替えられます。
- 味方ポケモンを選ぶと、味方の素早さラインと相手の素早さラインを比較します。
- パーティ情報では、敵6体に通る攻撃タイプと味方6体に通される攻撃タイプを平均倍率で表示します。
- 技、持ち物、特性の使用率は棒グラフ付きで表示します。

## 素早さライン

- 特化: 32振り実数値 x 1.1、小数点以下切り捨て
- 32振り: 性格補正なし + 32振り
- 無振り: 性格補正なし + 0振り
- 下降: 無振り実数値 x 0.9、小数点以下切り捨て
- 最速S: 特化実数値 x 1.5、小数点以下切り捨て

## 次に足すもの

- OBS映像から相手ポケモン名表示領域を切り出す
- OCR結果をポケモン名辞書で補正する
- Pokémon Champions向けの使用率CSV/JSONを読み込む
- 自分のパーティ6体と努力値調整を保存する

## データ更新

`scripts/update-data.js` はスクレイピング取得を追加するための土台です。

```bash
node scripts/update-data.js
```

現時点では `data/pokemon.json` と `data/champions.json` のメタデータ更新のみ行います。安定した取得元が決まったら、このスクリプトに取得・整形処理を追加します。
